## Danh sách các Labs đã thực hiện:

- **Lab01:** Thiết lập cơ sở dữ liệu trên MongoDB Atlas, nạp dữ liệu mẫu (`sample_mflix`) và thực hiện các truy vấn cơ bản.
- **Lab02:** Khởi tạo dự án Backend Node.js, cài đặt môi trường (Express), và xây dựng kiến trúc MVC (DAO, Controller) để tạo API truy xuất danh sách phim.
- **Lab03:** Hoàn thiện Backend: xây dựng các API quản lý Review (Thêm, Xóa, Sửa) và API lấy chi tiết phim.
- **Lab04:** Khởi tạo dự án Frontend với ReactJS, cài đặt UI framework (React-Bootstrap), và thiết lập thanh điều hướng cùng định tuyến (React Router DOM).
- **Lab05:** Kết nối Frontend và Backend (Axios), xây dựng giao diện hiển thị danh sách phim, tính năng tìm kiếm, và hiển thị chi tiết phim cùng các reviews.
